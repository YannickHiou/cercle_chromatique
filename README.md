
# Cercle chrosmatique

![Cercle Chromatique revisité](cercle.png)

Ce cercle chromatique réorganise celui de Johannes Itten : les six couleurs tertiaires occupent la grande couronne, les trois couleurs secondaires la couronne intérieure et les trois couleurs primaires le centre. Graphiquement, chaque couleur tertiaire est composée d’une couleur primaire et d’une couleur secondaire, et chaque couleur secondaire est composée de deux primaires. La représentation est ainsi  simplifiée pour donner une vue d’ensemble claire.
